## Getting Started

```bash
git clone git@github.com:LucasOAssuncao/Project-NG.git
```

### Install dependencies:

#### With Docker:

```bash
docker-compose up --build
```

#### Without Docker:

```bash
npm install
```

### Run server

```bash
cd backend/ 
npm start
```

and

```bash
cd frontend/ 
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
